Correlation of Media to Violence

ABOUT THE VISUALIZATION

This visualization was created as a tool with which one can explore the relationships between police reports, news reports, and social media shares of crime in the Seattle area from January 18th to February 7th. We began the visualization with the hypothesis that society’s perception of crime would more closely mirror the representation of crime crafted by the media than that of the police reports and then created a tool that allows users to create their own visualization and explore their own hypothesis regarding the public perception of crime and crime in the media and subsequently refute or support their belief.

Developed by Jax and Charles

MAKE YOUR OWN

1. Download the code.

          Here’s a link to our code.

2. Create your own JSON files.

          Our visualization uses JSON files to read the data, thus, to use this tool, you must create your own JSON files. We created three different JSON files: AssaultRobberyDS.json, NewsArticles.json, and SharesArticles.json.

          To create your own JSON file, add each element as a child under the appropriate topic - homicide, assault, or robbery.

          Here is how the news article JSON is formatted:

                    "name": "Assault",

                              "children"

                                        [

                                                  {"location":"King County Court", "date":"1/20/2015"}

                                        ],

          Add one of those children for each article you recorded in your data with the appropriate information.

          For the social media shares JSON, each child tells you what type of share (Facebook vs. Twitter) instead of location and date.

                    [

                              {"share":"Facebook"}

                    ],

3. Change the locations of the JSON files in the code.

          The code needs to be able to locate the JSON files locally on you computer. Thus, you must change the locations of the JSON files in the index.js file to the locations on your computer so that the visualization can locate the files.

          On a mac, to figure out the path, select the file and click “get info.” There, it will tell you the location of the file. Paste this where instructed within the code.

4. Open “index.html” to see your visualization!

          Google Chrome, Safari, and Internet Explorer do not allow you to access files locally. For this visualizaiton to work, you must use Firefox.
